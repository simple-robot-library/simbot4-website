# 唯一标识 ID

<tldr>

<p>一个针对“唯一标识”的简单值包装类型，
用以屏蔽不同类型的ID之间的差异。</p>
<p><code>ID</code> 是多平台实现。</p>

</tldr>

## 唯一标识类型

`ID` 是一个密封类型，提供了如下几种类型的实现类型：

- `StringID`
- `UUID`
- `IntID`
- `LongID`
- `UIntID`
- `ULongID`

并可将它们简单的归类为：

- 字符串类型
- 数字类型
  - 有符号数字类型
  - 无符号数字类型
